The internet's fastest privacy-first YouTube playlist duration calculator and creation toolkit. Built for creators, students, and lifelong learners who need accurate timing without tracking or ads.

## 🚀 The Complete YouTube Toolkit

This site combines three powerful tools:
- **📊 Playlist Length Calculator** - Total duration, video count, average length, and speed matrix
- **🖼️ Thumbnail Downloader** - HD MaxRes thumbnails (1920×1080), SD, HQ, MQ
- **⏰ Deadline Planner** - Plan daily watch schedules at custom speeds to finish by your target date

## ✨ Key Features

### YouTube Playlist Calculator
- Calculate total duration of any public YouTube playlist
- Multiple playlists support
- Custom playback speeds (0.1x–10x)
- Video range selection
- Search & sort functionality
- Speed matrix showing time saved at different speeds

### YouTube Thumbnail Grabber
- Download thumbnails in all available resolutions
- **Note**: 4K thumbnails don't exist on YouTube; MaxRes (1080×720) is the maximum
- Direct from YouTube's CDN - no login, no watermarks
- Works for Shorts with appropriate resolution limits

### YouTube Deadline Planner
- Study planner for finishing playlists by deadlines
- Set custom speeds and multiple sessions per day
- Speed matrix comparing daily watch time at every speed
- Tracks progress and watch history locally

## 🛡️ Privacy-First Design

- **Zero tracking** - No Google Analytics, cookies, or third-party scripts
- **100% client-side** - All calculations run in your browser
- **No data collection** - Never collect playlist URLs, video titles, or watch history
- **No ads** - Completely free, forever
- **Open source** - Under MIT License
- **Local storage only** - Your data stays on your device

## 🔧 Technology Stack

Built with modern, privacy-respecting technologies:

- **Frontend**: Astro 5 (static site generation), TypeScript, Tailwind CSS 4
- **Core library**: Vanilla JavaScript
- **API**: YouTube Data API v3
- **Fonts**: Geist and Geist Mono variable fonts
- **Package management**: npm (Node.js 22+)

## 🔑 For Users

### Need Your Own YouTube API Key?
1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create a new project
3. Enable YouTube Data API v3
4. Create API key in Credentials
5. Paste into "Use your own API key" field
6. Free tier: 10,000 units/day (generous for personal use)

### Private & Unlisted Playlists
Connect YouTube via OAuth 2.0 to access private playlists, unlisted playlists, and liked videos. All credentials stay in your browser - never sent to servers.

### Mobile Usage
- Fully responsive design
- Add to home screen for app-like experience
- Touch-friendly interface
- Works on iOS Safari, Android Chrome, Firefox, etc.
